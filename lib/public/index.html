Congratulations! Your Hudda webserver is running!

Disabling this

To turn this off, change the key "webserver": { "enabled": true } to "webserver": { "enabled": false } in lib/settings/bot_settings.json

Adding a path in your plugin

To add a web path for your plugin, do this in your code (make sure it gets executed): 

Web.get '/mypath' do
  "Hello friends"
end
The web server is just a Sinatra app, so all Sinatra things apply. Just remember to add a Web in front of the HTTP methods.
